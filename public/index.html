Attributes

You can configure the look of the subscription formular to a certain degree. Also you are able to set some meta properties on every subscription. Use the following attributes:

  • data-public-key (required) : Public Key of the Listly List, enables to identify to which list the subscription belongs.
  • data-lang : Language code for the input fields placeholders, must be 'en' or 'de'
  • data-button-label : Text label that will appear on the 'Submit' button
  • data-button-bg-color the color of the 'submit' button
  • data-props properties that are saved together with the subscriber emails

    data-props can be used for example for A/B tests or identifing where the subscription happened. You are free however to set up any properties you want. Every property should contain in its key (name) the prefix 'prop-'. The property source would also be here as 'prop-source=value'. You can separate many properties by the sign '&', e.g prop-test1=value1&prop-test2=value2.

    A/B Test Example:

    data-props="prop-campaign=campaign1&prop-source=test2&prop-test_property=test3"

    or

    data-props="prop-source=landingpage1"
